# Xcyberex Portfolio Website

A futuristic 3D animated portfolio site designed for **Xcyberex**.  
This repository contains the implementation code based on the original design concept available on [Figma](https://www.figma.com/design/cp3OL2QAANiiGZiTvQoIAT/Xcyberex-Portfolio-Website-Design).

---

## Project Overview
- **Brand**: Xcyberex — a cyber‑tech inspired personal portfolio.
- **Design style**: Dark mode, neon accents (blue/purple), holographic 3D elements, and smooth animations.
- **Purpose**: Showcase projects, skills, and contact information in an immersive, interactive format.
- **Hosting**: Optimized for deployment on [Vercel](https://vercel.com).

---

## Features
- 3D Hero Section: Animated intro with tagline and interactive call‑to‑action.
- Projects Grid: 3D cards with hover animations (flip, rotate, zoom).
- Skills Section: Animated progress bars and radial charts.
- Background Effects: Particle animations and parallax scrolling for depth.
- Contact Form: Futuristic glowing inputs with animated send button.
- Responsive Design: Works seamlessly across desktop, tablet, and mobile.

---

## Tech Stack
- **Framework**: Next.js (React-based, optimized for Vercel)
- **Styling**: TailwindCSS for utility-first responsive design
- **3D & Animations**: Three.js / React Three Fiber for 3D visuals
- **Forms & Interactivity**: React Hook Form / Framer Motion
- **Deployment**: Vercel for CI/CD and hosting
